# 문서 안내

이 폴더는 개발자가 현재 프로젝트를 빠르게 파악하기 위한 최소 문서만 담습니다.

## 읽는 순서

1. `../HANDOFF.md`
2. `../README.md`
3. `vibe_coding_minimal_context_v1.md`
4. Supabase 배포나 권한 검증이 필요하면 `supabase_go_live_checklist.md`
5. 마이그레이션 적용 순서는 `../supabase/README.md`

## 문서 운영 원칙

- Markdown 문서는 한국어로 작성합니다.
- 현재 코드와 맞지 않는 과거 기획 문서는 이 전달본에 추가하지 않습니다.
- 구현 판단은 문서보다 현재 `src/` 코드와 Supabase migration을 우선합니다.
- 새 기능 기획서는 현재 기능 안정화 이후 별도 문서로 추가합니다.
